########
Overview
########

************
Requirements
************

The client requires `PHP >= 8.3 <http://php.net/>`__.

************
Installation
************

The recommended way to install ImboClient is with `Composer <http://getcomposer.org/>`__. Install Composer if you do not already have it on your system.

To install ImboClient you can run the following command:

.. code-block:: bash

    composer require imbo/imboclient:^3.0

After installation you need to require the autoloader generated by Composer:

.. code-block:: php

    require 'vendor/autoload.php';
